Key Functions

Function Description

spdk_ioat_get_dma_capabilities() Get the DMA engine capabilities.
spdk_ioat_submit_copy() Submit a DMA engine memory copy request.
spdk_ioat_submit_fill() Submit a DMA engine memory fill request.
