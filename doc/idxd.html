IDXD Driver

Public Interface

  • spdk/idxd.h

Key Functions

Function Description
spdk_idxd_probe() Enumerate the IDXD devices attached to the system and attach the userspace IDXD driver to them if desired.
spdk_idxd_submit_copy() Build and submit a accel engine memory copy request.
spdk_idxd_submit_compare() Build and submit a memory compare request.
spdk_idxd_submit_crc32c() Build and submit a memory CRC32-C request.
spdk_idxd_submit_dualcast Build and submit an accel engine dual cast copy request.
spdk_idxd_submit_fill() Build and submit a accel engine memory fill request.

Pre-defined configurations

The RPC idxd_scan_accel_engine is used to both enable IDXD and set it's configuration to one of two pre-defined configs:

Config #0: 4 groups, 1 work queue per group, 1 engine per group. Config #1: 2 groups, 2 work queues per group, 2 engines per group.
